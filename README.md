\# Parking Management System



A console-based Parking Management System developed in C++ using Object-Oriented Programming concepts.



\## Features



\- Park a vehicle

\- Display parked vehicles

\- Remove a vehicle

\- Parking lot capacity management

\- Menu-driven interface



\## Concepts Used



\- Classes and Objects

\- Encapsulation

\- Constructors

\- STL Vector

\- Linear Search

\- Dynamic Data Storage

\- Menu-Driven Programming



\## Project Structure



\- Vehicle Class

&#x20; - Stores vehicle details

&#x20; - Displays vehicle information



\- ParkingLot Class

&#x20; - Parks vehicles

&#x20; - Displays vehicles

&#x20; - Removes vehicles



\## Technologies



\- C++

\- Standard Template Library (STL)
